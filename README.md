## 🛠️ My Methodology & Toolkit

Every room is approached with a structured penetration testing methodology. Common tools used in these write-ups include:

*   **Reconnaissance:** `nmap`, `gobuster`, `dirbuster`
*   **Exploitation:** `Metasploit`, manual exploit modification, `Netcat`
*   **Privilege Escalation:** manual enumeration
*   **Post-Exploitation:** Cracking hashes with `John the Ripper` or `Hashcat`
